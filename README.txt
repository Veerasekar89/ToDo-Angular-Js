How To Use
==========
1. Open index.html from browser exepect IE8.
2. Fill add entry form and submit(Title, Description), After submiting Your entries.
   will load below the entry form.
3. In search box you can filter both Title and Description. Search query should be 3 characters or more
   For example,
	Entry A : name "meet a friend" and description "bugis mrt. 7.30pm"
	Entry B : name "swimming" and description "toa payoh swimming cplx. 8am"

	Type "fr" : whole list
	Type "friend" : Entry A
	Type 'swim'   : Entry B
	Type "shopping" : Empty
4. On Mouseover  in each row Assoicated description will shown in tooltip.
5. If you need to delete any entry you should mark as complete( Select checkbox).
